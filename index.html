1. Sentiment analysis using NLP

07/22 - 12/22

Our project focused on building a sentiment classification model to analyze customer opinions from airline-related tweets. The dataset contained real-time tweets labeled as positive, negative, or neutral. The key steps included data preprocessing using NLTK for noise removal, lemmatization, and stopword filtering. A Bag-of-Words (BoW) model with tri-gram tokenization was applied using CountVectorizer to convert textual data into numerical features. A Logistic Regression classifier was trained to predict sentiment from tweets. The model achieved an overall accuracy of 74%. Performance was evaluated using confusion matrix and classification metrics, highlighting strong performance on negative sentiment but lower recall for neutral and positive classes, indicating potential data imbalance or class separation challenges.
